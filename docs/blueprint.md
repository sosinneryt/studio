# **App Name**: Sense Oasis

## Core Features:

- Breathing Exercise Tool: A guided breathing exercise tool with visual cues to aid relaxation.
- Soundscape Player: A player featuring a curated library of calming audio tracks such as 'Gentle Rain' and 'White Noise'.
- Light Pattern Customizer: An interface allowing users to create and display soothing light patterns.
- Interactive Sensory Toys: A collection of simple, interactive digital toys and games for sensory exploration (e.g., a coloring book, a toy simulation).
- Adaptive Sensory Profile: User profile setup with a questionnaire to understand and adapt to the user's sensory preferences. Generates content tuned to that profile.
- Personalized Activity Recommendations: Suggest activities and tools tailored to the user's sensory profile as interpreted by an AI tool.
- Progress Visualization: A dashboard to visualize the user's activity and progress over time, displaying time spent on different activities.

## Style Guidelines:

- Clean, minimalist layout with ample whitespace to minimize visual clutter.
- Primary color: Muted lavender (#D0B8E8) to evoke calmness and tranquility.
- Background color: Very light lavender (#F8F5FA) for a soft, non-intrusive backdrop.
- Accent color: Soft, dusty rose (#E5A8B2) for highlights and interactive elements.
- Font: 'PT Sans' (humanist sans-serif) for all text; easy to read and gentle.
- Subtle animations and slow transitions to create a calming user experience.
- Simple, geometric icons with rounded edges for a gentle and approachable feel.
